# CalcKcal — カロリー計算アプリ

毎日の食事を記録して、カロリーと栄養バランス（PFC）をかんたんに管理できるモバイル向けWebアプリです。
ビルド不要・依存ライブラリなしのバニラ HTML / CSS / JavaScript で実装しており、`index.html` をブラウザで開くだけで動作します。データは端末内（localStorage）に保存されます。

## 使い方

```
# いずれかの方法で index.html を開くだけ
open index.html               # macOS
# もしくは簡易サーバ
python3 -m http.server 8000   # → http://localhost:8000
```

## 画面構成

下部タブ（ホーム／記録／履歴／マイページ）＋どの画面からも押せる「＋記録」FAB の構成です。

| 画面 | 内容 |
|------|------|
| **0. オンボーディング**（初回のみ） | ようこそ → プロフィール入力（年齢・性別・身長・体重・活動レベル・目的）→ 目標カロリー/PFC自動算出と手動調整 → 完了 |
| **1. ホーム** | 今日の総合評価（◎/△/×）、カロリーリング（残りkcal）、PFCバランスバー（緑＝適正/黄＝注意/赤＝過不足）、不足・過剰アラート、食事区分別リスト |
| **2. 記録** | 食事区分選択、検索／お気に入り・履歴／バーコード／写真タブ、食品検索リスト |
| **3. 食品詳細** | 数量入力（数値＋スライダー）、選んだ量での栄養（kcal/PFC）リアルタイムプレビュー、追加ボタン |
| **4. 履歴** | 日/週/月切替、カロリー摂取推移（PFC積み上げ棒グラフ＋目標ライン）、達成率サマリー、体重推移グラフ |
| **5. マイページ** | プロフィール編集、目標の再計算・手動調整、体重記録、リマインド設定、ヘルスケア連携、データリセット |

## 計算ロジック

- **基礎代謝(BMR)**: Mifflin-St Jeor 式
- **目標カロリー**: BMR × 身体活動レベル係数 + 目的別補正（維持±0／減量−400／増量+300 kcal）
- **PFC配分**: 目的別の比率からグラム換算（P・C=4kcal/g, F=9kcal/g）

## ファイル構成

```
index.html        画面シェル（ヘッダー / タブ / FAB）
css/styles.css    モバイルUIスタイル
js/data.js        食品データベース・栄養/目標計算
js/store.js       状態管理・localStorage 永続化
js/app.js         画面ルーティングと描画
```

> 表示される数値は一般的な栄養データに基づく目安であり、医療・栄養指導を代替するものではありません。
